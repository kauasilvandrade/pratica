Formatações de textos

Tag <STRONG>

A tag <STRONG>serve para deixar um palavra em negrito. Usamos isso para dar significado a algo.

Exemplo:

Knull é muito forte.

Tag <EM>

A tag <em> é usada para dar ênfase a alguma palavra deixando ela em itálico.

Exemplo:

Em HTML 5, o que anteriormente era chamado de conteúdoblock-level agora é chamado de conteúdo flow.

Tag <MARK>

Usamos a tag <mark> para marcar um texto.

Exemplo:

PENSA MARK!

Tag <SMAL>

A tag <small> é usado para deixar um texto pequeno.

Exemplo:

Esse texto esta pequeno.

Tag <DEl>

A tag <del> é usada para indicar um texto excluido mas não vai ser considerado.

Exemplo:

Todos vocês vão ser executados.

Tag <INS>

A tag <ins> é usada para dar uma ênfase e indicar que ele foi adicionado depois.

Exemplo:

O título de uma história é muito importante.

Tag <SUP>

A tag é <sup> usada para escrever coisas do tipo x2+3

Exemplo:

A2 + B2 = x

Tag <SUB>

A tag <sub> é usada para escrever coisas do tipo H2O

Exemplo:

CO2
